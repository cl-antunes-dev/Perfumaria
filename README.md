Projeto demonstrativo de e-commerce voltado ao segmento de perfumaria, desenvolvido para simular uma loja virtual moderna de perfumes, fragrâncias, kits e produtos de cuidados pessoais.

A aplicação apresenta catálogo de produtos, lançamentos, categorias, ofertas em destaque e chamadas para ação, com foco em navegação intuitiva, identidade visual sofisticada e experiência de compra agradável.

O projeto foi construído com HTML5, CSS3 e JavaScript, aplicando conceitos de responsividade, estruturação semântica, estilização de componentes e interatividade. Desenvolvido para fins de estudo, prática e composição de portfólio.
